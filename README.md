# Online Shopping Dashboard - Task 3

A **responsive and interactive dashboard** for an online shopping system. Built using HTML, CSS, JavaScript, and JSON, this dashboard displays data dynamically and provides a clean interface for users.  

---

## Features

- **Interactive Dashboard** – Displays key metrics and user data.  
- **Dynamic Data Loading** – Uses `data.json` to populate content.  
- **Responsive Design** – Works well on desktop and mobile devices.  
- **Custom Styling & Scripts** – CSS and JavaScript provide animations and interactivity.
